# Hospital Emergency Room Dashboard
---
### 📌 Project Overview

#### Built an interactive Power BI dashboard analyzing 9,216 emergency room patient records (April 2023 – October 2024) to monitor operational performance and patient trends.
---
# 🎯 Problem Statement

### Hospitals often lack a centralized system to track emergency room metrics such as wait times, admissions, referrals, and patient satisfaction, leading to challenges in operational planning.
---


# 🎯 Project Objective

#### To design a data-driven dashboard that tracks key emergency room KPIs and enables structured performance monitoring.
---
# ⚙ Data Preparation & Modeling

#### Cleaned and transformed raw data using Power Query
#### Created calculated measures using DAX
#### Designed structured data model with proper relationships
#### Built interactive filters and drill-through navigation
---
# 📊 Key KPIs

### Total Patients: 9,216
### Average Wait Time: 5.3 minutes
### Admission Rate: ~50%
### Patient Satisfaction Score: 4.99 / 10
---
# 🔍 Key Insights

#### Nearly 50% admission rate indicates high emergency case severity.
#### Highest visits from 30–39 age group.
#### Referrals concentrated in General Practice and Orthopedics.
#### Moderate satisfaction despite low wait time suggests service improvement opportunities.
---
# 🚀 Impact

#### Improved visibility of emergency room performance by centralizing KPIs, supporting better resource planning, and enabling data-driven decision-making
